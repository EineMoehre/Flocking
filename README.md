
### Dependencies:

- [GLFW 3](https://www.glfw.org/)  
- [GLAD](https://github.com/Dav1dde/glad)  
> glad could work without installing since I've added my own glad files in the src folder  
- [GLM](https://glm.g-truc.net/0.9.9/index.html)  

### How to run:

1. compile  
```bash
make all
```
2. run
```bash
./main.exec
```
